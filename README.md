# 🚀 Minha Jornada Fullstack - Isi.Flix

Bem-vindo(a) ao meu repositório de estudos! Aqui eu guardo toda a minha evolução como programador, desde os primeiros passos no HTML até o desenvolvimento de aplicações mais robustas com Angular e Java.

Este projeto faz parte do meu aprendizado no **Isi.Flix**, guiado pelo Professor Isidro.

---

## 📂 O que tem por aqui?

O repositório está organizado por tecnologias, facilitando a navegação:

### 🎨 Front-End (O visual)
*   **/front-end/htm, css, js**: Meus primeiros passos com a estrutura e estilo da web.
*   **/front-end/projeto/bootstrap**: Projetos práticos usando o framework Bootstrap para criar sites responsivos:
    *   **Crypto**: Uma interface moderna para o mundo das criptomoedas.
    *   **Pet**: Um site dedicado a serviços para pets.

### 💻 Java (A lógica e o Backend)
*   **/Java**: Exercícios de lógica de programação. Aqui você encontra resoluções de desafios do **Beecrowd** (BEE1001, BEE1018, etc.) e conceitos fundamentais como `TesteIF` e `Leitura` de dados.

### 🟦 TypeScript
*   **/TypeScript**: Estudos sobre a linguagem que traz "superpoderes" ao JavaScript, focando em Tipagem e Programação Orientada a Objetos (POO).

### 🅰️ Angular (O framework profissional)
*   **/projeto1**: Meu primeiro projeto completo em Angular!
    *   Estruturado com componentes: `cabecalho`, `conteudo`, `rodape` e muito mais.

## 🚀 Como rodar o projeto Angular?

Se você quiser ver o meu projeto Angular funcionando na sua máquina:

1. Entre na pasta do projeto:
   ```bash
   cd projeto1
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor:
   ```bash
   ng serve
   ```
4. Abra o navegador em `http://localhost:4200/`
